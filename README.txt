## 📌 Descripción

Este proyecto es un sitio web desarrollado como portafolio personal, con el objetivo de mostrar mis trabajos, habilidades técnicas y facilitar el contacto profesional.

Está diseñado como una **landing page moderna**, enfocada en la claridad, usabilidad y presentación visual.

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Render (deploy)
